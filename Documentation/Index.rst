TYPO3 Helpers
=============

**Reusable helper traits and utilities for TYPO3**, including a Microsoft Graph
mail conversion service that turns a Symfony ``SentMessage`` into a Graph-ready
message object — with full support for recipients, HTML/plain bodies, file
attachments and inline ``cid:`` images.
